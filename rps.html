Rock Paper Scissor Game

Rock Paper Scissor Game is Javascript based Game. With the help of DOM manipulation.

In this game there are two opponents Player and Computer. Each opponent has their score initially zero and for each win score increase.

The logic of this game is based on random generation of number between 1 to 3.

The player can select their choice by clicking Rock Paper Scissor button On the other hand computer choice can be generate randomly.

The comparison choices can be done by game logic.
