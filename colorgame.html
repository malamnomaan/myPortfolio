The Color Game

The Color Game is Javascript based Game. With the help of DOM manipulation. In this game the RGB value is generated randomly. You can guess the color shade on the basis of corrosponding RGB value and select the circle.

This Game has 2 modes i.e Easy and Hard.

In Easy Mode there are 3 circles and in Hard mode there are 6 circles in which 1 option is correct.
