The Flappy Bird Game

The Flappy Bird Game is Python based application. With the help of pygame module

The logic of this game is based on x and y co-ordinates. When the co-ordinates of bird and pipes are same then Game Over. You can pass the bird between upper and lower pipes.

This Game also contain Day and Night Mode
